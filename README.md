# 믿어주세요 개표국

풍자형 픽셀 방치 타이쿤 게임입니다. 개표국 시설을 키우고, 직원을 채용하고, 사건 카드에 대응하며 구역별 개표를 완료하는 모바일 세로형 PWA를 목표로 합니다.

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 현재 구현

- Phaser 3 + Vite 기반 모바일 세로 화면
- 표/해명/믿음 3자원 경제
- 시설 업그레이드와 구역별 시설 해금
- 직원 채용/강화와 생산 배율 반영
- 직원 희귀도, 스킬 해금, 직군별 카드 UI
- 랜덤 사건 카드와 선택지 보상/패널 UI
- 목표/업적/운영 기록
- 스테이지 진행률과 지역 완료 보상
- 감사 인장 프레스티지와 영구 업그레이드
- 시설별 픽셀 아이콘, 선택 강조, 업그레이드 가능 배지
- 자동 저장, 세이브 버전, 오프라인 보상
- PWA manifest와 서비스 워커 캐시 기본 구조
