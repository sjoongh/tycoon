# 개표국 — 밤샘 자율 미션 헌장

**목표:** 실제 운용 가능한 게임 앱. 실서비스 중인 동류 게임 대비 **품질 95%+ 동등**. 중간보고·질문·사용자 결정 요청 **없음** — 전부 자율 결정·실행, 목표 달성까지 반복.

**아트/톤:** 거지키우기(rags-to-riches) 류. 코지 파스텔 폐기.

## 7역할 루프 규약 (큰 루프 1회마다 전원 반드시 포함)
매 큰 루프(=한 단계 완성)마다 아래 7역할로 리뷰·검증·개발하고, 목표 미달이면 될 때까지 반복:
1. 시니어 게임 개발자 — 아키텍처·perf·버그·게임필
2. 게임 기획자 — 루프·밸런스·리텐션·실게임 대비
3. UI/UX — 정보구조·플로우·온보딩·접근성
4. 디자이너 — 비주얼·구도·톤·일관성
5. QA — 기능/시각/엣지 버그 (Playwright로 직접 구동)
6. 알파테스터 — 코어 루프가 실제로 작동/재미있나 (실플레이)
7. 베타테스터 — 장시간/리텐션/체감, 실게임과 95% 비교

검증 수단: 항상 Playwright로 **직접 플레이·스샷**(phone 390x844 + desktop). 사용자에게 스샷 요청 금지.

- P2 게임필/주스: 탭·수입·마일스톤·사운드(가능시)·온보딩 완성
- P3 밸런스: 1분/10분/1시간/하루 곡선 튜닝(실플레이 데이터 기반)
- P4 콘텐츠/리텐션: 일일목표·업적·프레스티지 깊이·사건
- P5 출시 준비: PWA·세이브 견고화·설정·QA 하드닝
- 최종: 7역할 종합 검증, 95% 기준 충족까지 반복
